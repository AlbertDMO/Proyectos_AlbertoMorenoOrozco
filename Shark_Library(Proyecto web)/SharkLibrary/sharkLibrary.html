Shark Library, es un directorio de Software libre. Nos basamos en la idea de compartir Software libre de manera sencilla.

Tratamos de ofrecer una interfaz amigable para el usuario.

Creemos que através de los comentarios, se puede difundir y expresar opiniones acerca de un Software en específico.
